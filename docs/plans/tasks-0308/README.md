# Tasks 0308：OpenClaw 社区热点内容创作任务

更新时间：2026-03-08

本目录下的每个文件都是一个可以直接分配给 sub-agent 的内容创作任务。

## 使用方式

- 将单个任务文件全文作为 prompt 提供给 sub-agent。
- 执行前，sub-agent 应先阅读任务中指定的内部上下文与外部讨论源。
- 产出应直接面向 CoClaw 内容体系，避免泛泛新闻稿或社区八卦文。
- 所有 task 的最终交付稿件都必须使用英文（标题、description、正文均为英文）。
